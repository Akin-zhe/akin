# Akin — Product Designer Portfolio

A modern, responsive portfolio website for Akin, an experienced Product Designer with 18+ years in the field.

## 🎨 Overview

This is a professional portfolio showcasing design work, services, and expertise in UI/UX design, application design, and web design.

## ✨ Features

- **Responsive Design** — Works seamlessly on desktop and mobile devices
- **Hero Section** — Eye-catching introduction with call-to-action buttons
- **Services Section** — Highlights key design services offered
- **About Section** — Professional background and statistics
- **Tools Showcase** — Display of design tools and software used
- **Animated Elements** — Smooth animations and interactive components
- **Contact Section** — Easy way for clients to get in touch
